# Glossário — Higilabor

## Siglas e Termos Técnicos SST
| Termo | Expansão / Significado |
|-------|----------------------|
| SST | Saúde e Segurança do Trabalho |
| NR | Norma Regulamentadora (ex: NR-1, NR-10, NR-12) |
| NR-1 psicossocial | Exigência nova: PGR deve contemplar riscos psicossociais a partir de maio/2026 — principal gatilho comercial do momento |
| PGR | Programa de Gerenciamento de Riscos |
| PCMSO | Programa de Controle Médico de Saúde Ocupacional |
| LTCAT | Laudo Técnico das Condições Ambientais do Trabalho |
| Laudo insalubridade | Laudo que caracteriza agentes insalubres no ambiente de trabalho |
| FAP | Fator Acidentário de Prevenção — multiplica a alíquota de RAT, impacta folha de pagamento |
| RAT | Risco Ambiental do Trabalho (alíquota previdenciária) |
| eSocial | Sistema digital obrigações trabalhistas/tributárias (SEFIP, GFIP, CAGED etc.) |
| GFIP | Guia de Recolhimento do FGTS e Informações à Previdência Social |
| GPS | Guia da Previdência Social |
| DARF | Documento de Arrecadação de Receitas Federais |
| Simples | Simples Nacional — regime tributário da Higilabor |
| FGTS | Fundo de Garantia por Tempo de Serviço |
| UEP | Unidade de Esforço de Produção — capacidade operacional (650 UEP/mês) |
| Perícia trabalhista | Serviço de perícia técnica em processos judiciais trabalhistas |

## Termos Financeiros e Comerciais
| Termo | Significado |
|-------|------------|
| CR | Contas a Receber — carteira de títulos/parcelas a receber |
| CP | Contas a Pagar — obrigações e saídas |
| DRE | Demonstrativo de Resultado do Exercício (gerencial) |
| MRR | Monthly Recurring Revenue — receita mensal recorrente de contratos |
| Retainer | Contrato mensal recorrente (oposto de avulso) |
| Piso | Preço mínimo defensável por serviço: custo real + overhead + tributo + margem 20% |
| Ticket médio | Receita média por venda lançada |
| Margem bruta | Receita líquida menos custos diretos |
| EBITDA gerencial | Resultado antes de juros, impostos, depreciação — proxy gerencial |
| Pipeline | Funil comercial ativo de oportunidades |
| Follow-up | Acompanhamento ativo de proposta ou negociação |
| Aging | Faixa de vencimento dos títulos em aberto (ex: 1–30 dias, 31–60 dias) |
| Inadimplência | Títulos vencidos não recebidos |
| ERP | O sistema de gestão (atualmente a planilha 5 blocos multi-mês) |
| 5 blocos | Planilha xlsx com CR, CP, Fluxo de Caixa, Indicadores, DRE, Consolidado |

## Pacotes Comerciais
| Pacote | O que inclui |
|--------|-------------|
| Essencial | PGR + PCMSO base para pequenas empresas |
| Financeiro/FAP | Foco em redução do FAP e economia previdenciária recorrente |
| Premium Patrimonial | Gestão de risco patrimonial e responsabilidade civil |
| Psicossocial | NR-1 psicossocial — urgência regulatória maio/26 |
