# Reprecificacao Higilabor

Ferramenta HTML para apoiar decisoes de preco minimo, margem alvo e diagnostico rapido de contratos.

## Como usar

1. Abra `index.html` no navegador.
2. Ajuste os controles de custo, tributos, margem e sobretaxas.
3. Use a aba de diagnostico para validar riscos de margem.

- O arquivo atual foi mantido como HTML unico para facilitar copia e embed.
